# Servidor e cliente

📌 Descrição
Projeto em Node.js com Express e Socket.IO que cria comunicação em tempo real entre servidor e clientes.

🚀 Tecnologias
- Node.js
- Express
- Socket.IO
- JavaScript

🎯 Funcionalidades
- Comunicação em tempo real com WebSockets.
- Envio e recepção de mensagens ao vivo.
- Estrutura de servidor e clientes conectados.

🖥️ Demonstração
Link do projeto online:
https://seu-link-aqui.com

📸 Imagens

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `node servidor.js`
- Abra o navegador e acesse o servidor

📚 Aprendizados
Aprendi a usar Socket.IO para implementar comunicação em tempo real em uma aplicação Node.js.
