# Hashtaurante

📌 Descrição
Aplicação front-end em React e Vite que apresenta um cardápio de restaurante com componentes e navegação.

🚀 Tecnologias
- React
- Vite
- JavaScript
- CSS

🎯 Funcionalidades
- Exibe itens do cardápio de restaurante.
- Uso de componentes React para renderizar dados.
- Estrutura de projeto moderna com Vite.

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `npm run dev`
- Abra o endereço exibido pelo Vite no navegador

📚 Aprendizados
Aprendi a estruturar um projeto React com Vite, criar componentes e trabalhar com estilos em uma aplicação moderna.
