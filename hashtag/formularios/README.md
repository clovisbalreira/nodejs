# Formulários

📌 Descrição
Projeto Node.js com Express e validação de formulários usando `joi` para um fluxo de cadastro ou formulários web.

🚀 Tecnologias
- Node.js
- Express
- joi
- JavaScript

🎯 Funcionalidades
- Validação de dados no backend.
- Criação de rotas para processamento de formulários.
- Tratamento de requisições HTTP com Express.

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `node app.js`
- Abra o navegador e acesse a aplicação
