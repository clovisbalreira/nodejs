# Repositórios de linguagem

## Projetos em Node.js

📌 Descrição
Repositório com exercícios e projetos feitos em Node.js.

🧠 Conteúdo
- Aplicações web com Express
- APIs e back-end com MongoDB e SQLite
- Projeto React + Vite
- WebSocket com Socket.IO
- Integração com API externa usando Axios
- Uso de TypeScript, Prisma e validação com Zod

### Projetos

- `alura/base-de-conhecimento`
  - Gerador de base de conhecimento usando `dotenv`.
  - Projeto Node.js simples para gerar conteúdo com scripts.

- `alura/instalike/back-end`
  - Backend em `Express` com `MongoDB` e upload de arquivos com `multer`.
  - Estrutura de API de rede social com rotas e persistência.

- `hashtag/aula03/hashtaurante`
  - Aplicação front-end em `React` com `Vite`.
  - Projeto de cardápio / restaurante.

- `hashtag/aula04`
  - Servidor em `Express` com `Socket.IO`.
  - Projeto de comunicação em tempo real.

- `hashtag/aula05`
  - Projeto Node.js com scripts e exercícios básicos.

- `pessoal/nodejs`
  - Projeto em `Express` com `dotenv` e conexão `mysql2`.

- `pessoal/cripto`
  - Projeto que consome APIs externas com `axios`.
  - Exemplo de chamada a serviços de criptomoedas.

- `pessoal/bancoDados`
  - Projeto com `sqlite` e `sqlite3`.
  - Exercícios de banco de dados local.

- `rocketseat/gerenciamento-inscricoes`
  - Projeto em `TypeScript` com `Fastify`, `Prisma` e `Zod`.
  - API de gerenciamento de inscrições e eventos.

▶️ Como rodar

1. Abra o terminal na pasta do projeto desejado.
2. Execute `npm install` para instalar as dependências.
3. Inicie o projeto:
   - `npm run dev` quando disponível.
   - `node server.js`, `node app.js`, ou `node index.js` para projetos Node.js sem script de execução.
   - `npm run build` e `npm run preview` para projetos `Vite`.

> Dica: alguns projetos usam `type: module` ou `TypeScript`, então verifique o arquivo `package.json` de cada pasta para o comando exato de execução.
