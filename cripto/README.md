
# Cripto

Este é um projeto em Node.js para explorar funcionalidades relacionadas à criptografia.

## Funcionalidades

- Geração de hashes.
- Criptografia e descriptografia de dados.
- Manipulação de algoritmos de criptografia.

## Requisitos

- Node.js (versão 14 ou superior)
- npm (ou yarn)

## Instalação

1. Clone este repositório:
    ```bash
    git clone https://github.com/clovis-balreira/cripto.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd cripto
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```

## Uso

Execute o projeto com o comando:
```bash
node index.js
```
