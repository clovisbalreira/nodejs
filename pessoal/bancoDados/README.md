# BancoDados

📌 Descrição
Projeto Node.js para gerenciar operações de banco de dados local com SQLite e `sqlite3`.

🚀 Tecnologias
- Node.js
- SQLite
- sqlite3
- JavaScript

🎯 Funcionalidades
- Conexão com banco de dados SQLite.
- Execução de consultas e manipulação de dados.
- Criação e leitura de registros locais.

🖥️ Demonstração
Link do projeto online:
https://seu-link-aqui.com

📸 Imagens

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `node app.js`
- Observe os dados sendo gravados no arquivo `banco.db`

📚 Aprendizados
Aprendi a trabalhar com Banco de Dados SQLite no Node.js e a usar módulos de banco para persistência local.
