# Cripto

📌 Descrição
Projeto Node.js que consome APIs externas usando `axios` para trabalhar com dados de criptomoedas.

🚀 Tecnologias
- Node.js
- JavaScript
- axios

🎯 Funcionalidades
- Faz requisições a serviços externos.
- Processa e exibe dados de criptomoedas.
- Estrutura simples de API e manipulação de resposta.

🖥️ Demonstração
Link do projeto online:
https://seu-link-aqui.com

📸 Imagens

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `node index.js`

📚 Aprendizados
Aprendi a usar o `axios` para consumir APIs e tratar dados retornados em um projeto Node.js.
