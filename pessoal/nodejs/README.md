# Projeto Node.js

📌 Descrição
Aplicação backend em Node.js com Express e conexão a banco de dados MySQL via `mysql2`.

🚀 Tecnologias
- Node.js
- Express
- MySQL
- dotenv

🎯 Funcionalidades
- Configuração de servidor Express.
- Conexão com banco de dados MySQL.
- Rotas para manipulação de dados.

🖥️ Demonstração
Link do projeto online:
https://seu-link-aqui.com

📸 Imagens

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Configure o arquivo `.env` com as credenciais do banco
- Execute `node app.js`

📚 Aprendizados
Aprendi a configurar a conexão com MySQL em Node.js e a construir um backend simples com Express.
