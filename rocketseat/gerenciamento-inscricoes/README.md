# Gerenciamento de Inscrições

📌 Descrição
Projeto em Node.js e TypeScript para gerenciar inscrições em eventos, com backend Fastify e banco de dados gerenciado por Prisma.

🚀 Tecnologias
- Node.js
- TypeScript
- Fastify
- Prisma
- SQLite
- Zod

🎯 Funcionalidades
- Cadastro de eventos e participantes.
- Validação de dados com Zod.
- Persistência de informações usando Prisma e SQLite.

▶️ Como rodar
- Clone o repositório
- Execute `npm install`
- Execute `npm run dev`
- Abra o endereço exibido no terminal

📚 Aprendizados
Aprendi a criar APIs com Fastify, usar TypeScript em backend e integrar Prisma com SQLite para persistência de dados.
